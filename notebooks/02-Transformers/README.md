In these projects, it is possible to build a Transformer model **from scratch** to better understand its internal components and training process.

| Project | Description | Resource |
| :--- | :--- | :--- |
| **Mini GPT Char Transformer** | Demonstrates how a Transformer model is built from scratch at the character level. | Based on this video: https://www.youtube.com/watch?v=VMj-3S1tku0 |
| **Genesis Transformer** | Shows how to build and train a character-level Transformer using the *Book of Genesis* from the Bible as the training dataset. | Based on this video: https://www.youtube.com/watch?v=VMj-3S1tku0 |
